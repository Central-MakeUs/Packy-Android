### 기술 스택

- **Compose**: UI 구축에 Jetpack Compose 사용
- **Ktor**: Kotlin 기반 비동기 웹 프레임워크

### 아키텍처

- **MVI (Model-View-Intent)**: UI 상태 관리 및 비즈니스 로직 분리
